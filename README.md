# 🏎️ F1 Pit Stop Performance Comparison

## Overview

This project analyzes Formula 1 pit stop durations and compares the performance of multiple drivers using historical race data. The analysis focuses on pit lane efficiency and consistency among elite Formula 1 drivers.
